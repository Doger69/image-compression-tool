# Image Compression Tool

A desktop app that compresses images using three different methods.

- **JPEG** — standard quality-based compression
- **RLE** — Run-Length Encoding for simple images
- **Huffman Coding** — lossless compression using frequency analysis

## How to Run

Make sure Java is installed, then:

```bash
javac *.java
java CompressionGUI
```

Built with Java and Java Swing.
